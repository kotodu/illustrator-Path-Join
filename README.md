# illustrator-Path-Join
- An extension to join paths on Adobe illustrator CC.
- Adobe Illustratorで2つのパスの交点を補完し、結合するスクリプトです
- [GitHub](https://github.com/kotodu/illustrator-Path-Join/)

# 特徴
1. ボタン1つですぐ結合します
2. 交点を補完した上で、端の余った部分は削除します
3. 2つのパスでない場合、結合しません
